## Requirements:

	Anaconda

	Pytorch > 1.8


## Testing Images
	Keep Testing Images in "dataset" folder.

## Checkpoints:
	Keep the checkpoints in "./checkpoints/"

## For testing on any database:
	1. Run the "test.py" file as python test.py
	2. The results will be stored in "results/" folder
